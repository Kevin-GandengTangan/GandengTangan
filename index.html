Introduction

This document will help you to connect with GandengTangan Open API. Registered partner will be able to login/logout, register borrower, submit proposal, check proposal status, check list of available program, and send loan proposal agreement. If you need help, please contact Gandeng Tangan Development Team for further guidance.

This documentation aims to provide all the information you need to work with GandengTangan Open API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Base URL

http://dev-api-partnership.gandengtangan.com/

Authenticating requests

This API is authenticated by sending an Authorization header with the value "Bearer {YOUR_ACCESS_TOKEN}".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

You can retrieve {YOUR_ACCESS_TOKEN} by login to application. See how to login in the Account Authorization menu.

Account Authorization

APIs for managing account authorization.

Login to Application

In order to access GandengTangan Open API, you need to login first.

This endpoint lets you login to the application.
Example request:
import requests
import json

url = 'http://dev-api-partnership.gandengtangan.com/user/login'
payload = {
    "username": "gandengtangan",
    "password": "1234qweR"
}
headers = {
  'Content-Type': 'application/json'
}

response = requests.request('POST', url, headers=headers, json=payload)
response.json()
$client = new \GuzzleHttp\Client();
$response = $client->post(
    'http://dev-api-partnership.gandengtangan.com/user/login',
    [
        'headers' => [
            'Content-Type' => 'application/json',
        ],
        'json' => [
            'username' => 'gandengtangan',
            'password' => '1234qweR',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));
const url = new URL(
    "http://dev-api-partnership.gandengtangan.com/user/login"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "username": "gandengtangan",
    "password": "1234qweR"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
curl --request POST \
    "http://dev-api-partnership.gandengtangan.com/user/login" \
    --header "Content-Type: application/json" \
    --data "{
    \"username\": \"gandengtangan\",
    \"password\": \"1234qweR\"
}"

Example response (200, success):


{
    "code": 200,
    "description": "Successfully login!",
    "results": {
        "access_token": "{YOUR_ACCESS_TOKEN}",
        "token_type": "bearer",
        "expires_in": 3600
    }
}

Example response (400, failed):


{
    "code": 400,
    "description": "Failed!",
    "results": {
        "login": "Unauthorized"
    },
    "errorMesage": "Username and password do not match!"
}

Refresh Token

requires authentication

This endpoint lets you refresh your token.
Example request:
import requests
import json

url = 'http://dev-api-partnership.gandengtangan.com/user/refresh-token'
params = {
  'token': '{YOUR_ACCESS_TOKEN}',
}
headers = {
  'Authorization': 'Bearer {YOUR_ACCESS_TOKEN}',
  'Content-Type': 'application/json'
}

response = requests.request('GET', url, headers=headers, params=params)
response.json()
$client = new \GuzzleHttp\Client();
$response = $client->get(
    'http://dev-api-partnership.gandengtangan.com/user/refresh-token',
    [
        'headers' => [
            'Authorization' => 'Bearer {YOUR_ACCESS_TOKEN}',
            'Content-Type' => 'application/json',
        ],
        'query' => [
            'token'=> '{YOUR_ACCESS_TOKEN}',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));
const url = new URL(
    "http://dev-api-partnership.gandengtangan.com/user/refresh-token"
);

const params = {
    "token": "{YOUR_ACCESS_TOKEN}",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Authorization": "Bearer {YOUR_ACCESS_TOKEN}",
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());
curl --request GET \
    --get "http://dev-api-partnership.gandengtangan.com/user/refresh-token?token=%7BYOUR_ACCESS_TOKEN%7D" \
    --header "Authorization: Bearer {YOUR_ACCESS_TOKEN}" \
    --header "Content-Type: application/json"
